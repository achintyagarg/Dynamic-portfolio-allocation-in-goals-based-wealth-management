## Setup

### 1. Install Python dependencies

Run this once from anywhere. All packages in a single command:

```powershell
pip install stable-baselines3 gymnasium shiny shinywidgets shinyswatch plotly numpy pandas scipy matplotlib loguru aenum rich traceback-with-variables beautiful-traceback msgspec polars pyarrow pydantic great-tables plotnine yfinance xlsxwriter cvxpy skfolio azapy optimalportfolios babel
```

> **Note:** `stable-baselines3` will automatically install PyTorch as a dependency.

### 2. Generate training results

Run from the **project root** (the folder containing `run_training.py`):

```powershell
python run_training.py
```

This trains a PPO agent for 4 investor scenarios and saves results to `results/all_results.json`. Takes approximately 10–15 minutes. Must complete before the Strategy Comparison and RL Efficiency dashboard tabs will show data.

## Running the Dashboard

> **Critical:** Always run from the `Viz-QWIM-2026\` directory, not from inside `src\dashboard\`. The app uses `from src.xxx` imports that require the working directory to be `Viz-QWIM-2026\`.

```powershell
cd Viz-QWIM-2026
python -m shiny run src/dashboard/main_App.py
```

Then open your browser to `http://localhost:8000`.

### Why `python -m shiny` and not `shiny run`?

When installing packages with `pip` into the user directory (`--user` installs), the `shiny.exe` CLI may not be on your `PATH`. `python -m shiny` always works regardless of PATH configuration.

## Dashboard Tabs

| Tab | Description |
|-----|-------------|
| **Data Explorer** | Efficient frontier, 15-portfolio action space, asset correlation heatmap, simulated GBM wealth paths for VTBIX / VGTSX / VTSMX |

| Scenario | W₀ | Goal | Horizon | Cash Flow |
|----------|----|------|---------|-----------|
| Base case | $100K | $200K | 10 yr | $0/yr |
| Young investor | $50K | $500K | 30 yr | +$5K/yr |
| Retirement | $100K | $121.4K | 30 yr | +$15K/yr → -$50K/yr |

## Common Errors

### `shiny` not recognized
```
'shiny' is not recognized as the name of a cmdlet...
```
**Fix:** Use `python -m shiny run ...` instead of `shiny run ...`

---

### `ModuleNotFoundError` on startup
```
ModuleNotFoundError: No module named 'shinyswatch'
```
**Fix:** Re-run the full install command from the Setup section above.

---

### Import errors when running from `src\dashboard\`
```
ModuleNotFoundError: No module named 'src'
```
**Fix:** You must `cd` to `Viz-QWIM-2026\` first, then run `python -m shiny run src/dashboard/main_App.py`. Do not run `python main_App.py` from inside `src\dashboard\`.

---

### Strategy Comparison / RL Efficiency tabs show warning
```
Results not yet available. Please run python run_training.py...
```
**Fix:** Run `python run_training.py` from the project root. Wait for it to finish (~10–15 min) and confirm `results/all_results.json` was created.
